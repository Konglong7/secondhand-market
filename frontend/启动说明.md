# 二手交易平台前端启动说明

## 项目概述
基于 Vue 3 + Vite + Element Plus 的二手交易平台前端项目

## 技术栈
- Vue 3 (Composition API)
- Vite 8.0
- Element Plus 2.13
- Vue Router 4.6
- Pinia 3.0
- Axios 1.14

## 安装依赖
```bash
npm install
```

## 启动开发服务器
```bash
npm run dev
```
访问地址：http://localhost:5173

## 构建生产版本
```bash
npm run build
```

## 预览生产构建
```bash
npm run preview
```

## 核心功能

### 用户端
1. 用户注册/登录
2. 商品浏览（首页、列表页、详情页）
3. 商品搜索和筛选
4. 商品发布/编辑
5. 商品收藏
6. 订单管理（创建、支付、发货、收货）
7. 个人中心（个人信息、地址管理、我的发布、我的收藏）
8. 在线聊天（WebSocket）

### 管理端
1. 管理员登录
2. 数据统计
3. 用户管理（冻结、封禁）
4. 商品管理（审核）
5. 订单管理

## API配置
后端API地址配置在 `vite.config.js` 中：
```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true
    }
  }
}
```

## 注意事项
1. 确保后端服务已启动（默认端口8080）
2. 图片上传功能需要后端文件上传接口支持
3. WebSocket聊天功能需要后端WebSocket服务支持
4. 管理员登录使用独立的token存储（adminToken）
