# 二手交易平台功能需求文档

## 项目概述

**项目名称：** 二手交易平台（Web版）
**项目类型：** 类闲鱼二手交易平台
**技术栈：** Spring Boot + Vue 3 + MySQL + Redis
**开发阶段：** MVP（最小可行产品）
**文档版本：** v1.0
**更新日期：** 2026-03-30

---

## 一、用户模块

### 1.1 用户注册
- 支持手机号注册
- 支持邮箱注册
- 发送验证码（短信/邮件）
- 密码强度校验
- 用户协议确认

### 1.2 用户登录
- 手机号/邮箱 + 密码登录
- 验证码快捷登录
- 记住登录状态（7天/30天）
- JWT Token 认证

### 1.3 个人信息管理
- 修改头像（图片上传）
- 修改昵称
- 修改个人简介
- 实名认证（姓名 + 身份证号）
- 修改密码
- 绑定/解绑手机号/邮箱

### 1.4 收货地址管理
- 新增收货地址
- 编辑收货地址
- 删除收货地址
- 设置默认地址
- 地址列表查询

### 1.5 我的中心
- 我的发布（在售/已售/已下架）
- 我的购买（全部订单/待付款/待发货/待收货/已完成）
- 我的收藏
- 我的评价（收到的/给出的）
- 信用评分展示

---

## 二、商品模块

### 2.1 商品发布
- 上传商品图片（最多9张）
- 选择商品分类（二级分类）
- 填写商品标题（必填，50字以内）
- 填写商品描述（必填，500字以内）
- 设置商品价格（必填）
- 选择成色（全新/99新/95新/9成新/8成新及以下）
- 选择交易方式（仅邮寄/仅自提/都可以）
- 填写商品位置（省市区）
- 设置库存数量（默认1）

### 2.2 商品列表
- 首页推荐（综合排序）
- 分类浏览（一级/二级分类）
- 关键词搜索
- 筛选条件（价格区间/成色/地区）
- 排序方式（综合/最新/价格升序/价格降序）
- 分页加载

### 2.3 商品详情
- 商品图片轮播
- 商品基本信息（标题/价格/成色/位置）
- 商品详细描述
- 卖家信息（头像/昵称/信用分/在线状态）
- 相似商品推荐
- 商品收藏/分享
- 立即购买/联系卖家按钮

### 2.4 商品管理
- 编辑商品信息
- 上架/下架商品
- 删除商品
- 查看商品浏览量
- 商品状态（在售/已售/已下架）

### 2.5 商品收藏
- 收藏商品
- 取消收藏
- 收藏列表查询
- 收藏商品状态更新提醒

---

## 三、交易模块

### 3.1 下单购买
- 选择收货地址
- 确认商品信息
- 填写买家留言
- 生成订单（锁定库存）
- 订单超时自动取消（30分钟）

### 3.2 模拟支付
- 支付方式选择（支付宝/微信/余额）
- 模拟支付页面
- 模拟支付回调（成功/失败）
- 支付成功通知

### 3.3 订单管理
- 订单列表（全部/待付款/待发货/待收货/已完成/已取消）
- 订单详情查询
- 订单状态流转
  - 待付款 → 待发货
  - 待发货 → 待收货
  - 待收货 → 已完成
- 取消订单（待付款状态）
- 申请退款（待发货/待收货状态）

### 3.4 发货管理（卖家）
- 填写物流信息（快递公司/快递单号）
- 确认发货
- 模拟物流跟踪

### 3.5 收货管理（买家）
- 查看物流信息
- 确认收货
- 自动确认收货（发货后7天）

### 3.6 退款流程
- 买家申请退款（填写原因）
- 卖家同意/拒绝退款
- 退款成功（原路返回）
- 退款失败（继续交易）

---

## 十一、非功能需求

### 11.1 性能要求
- 首页加载时间 < 2秒
- 接口响应时间 < 500ms
- 支持并发用户数 > 1000

### 11.2 安全要求
- 密码加密存储（BCrypt）
- JWT Token 认证
- 接口防刷限流
- SQL注入防护
- XSS攻击防护
- 敏感信息脱敏

### 11.3 可用性要求
- 系统可用性 > 99%
- 数据备份机制
- 异常日志记录

## 十三、技术实现要点

### 13.1 后端技术栈
```
Spring Boot 2.7+
Spring Security + JWT
MyBatis Plus
MySQL 8.0
Redis 7.0
WebSocket / Netty（聊天）
OSS / 本地存储（图片）
```

### 13.2 前端技术栈
```
Vue 3 + TypeScript
Vite
Element Plus
Pinia
Vue Router
Axios
Socket.io-client
```

### 13.3 部署方案
```
Docker + Docker Compose
Nginx 反向代理
MySQL 主从复制（可选）
Redis 持久化
```

## 十五、接口设计规范

### 15.1 RESTful API 规范
- GET：查询
- POST：新增
- PUT：修改
- DELETE：删除

### 15.2 统一响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 15.3 错误码定义
- 200：成功
- 400：参数错误
- 401：未登录
- 403：无权限
- 404：资源不存在
- 500：服务器错误
